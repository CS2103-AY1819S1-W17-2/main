== Overview

modsUni is a desktop timetable management application created by students for students. It incorporates a simple GUI with majority of its user interactions occuring in its in-built CLI.

== Summary of contributions

* *Major enhancement*: **Generating** of a student's schedule was added.
** What it does: It allows students to generate a list of modules to take in each semester.
** Justification: This feature allows students to easily plan their candidature and it is also one of the
core features of modsUni.

* *Minor enhancement*: Loading in of modules as the database from a `XML` file.
